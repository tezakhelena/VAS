
# VAS
Projekt iz kolegija višeagentni sustavi

README – Pokretanje projekta
Link do projekta: https://github.com/tezakhelena/VAS.git
Za pokretanje aplikacije potrebno je imati instalirano:
•	Python 3
•	pip
•	Docker Desktop
Projekt koristi:
•	SPADE
•	Prosody (XMPP poslužitelj)
•	XMPP
Aktivacija virtualnog okruženja:
python -m venv venv
source venv/bin/activate

Instalacija spade:

pip install spade

Prosody XMPP poslužitelj pokreće se pomoću Docker Desktop aplikacije

Pokretanje skripte naredbom python main.py
